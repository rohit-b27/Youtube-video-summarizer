A web application that takes a YouTube video URL and generates a concise summary of its content. The app attempts to fetch a transcript if available or, if none is provided, transcribes the video’s audio using Whisper. It then leverages OpenAI’s LLMs (like GPT-4) to summarize the transcript. Optional integration with the YouTube Data API allows contextual enhancements to the summary using the video’s title, description, and tags.
